# Linux
명령어
---
whoami : 현재 유저명 출력  

pwd : 현재 디렉토리 출력  

ls : 파일 리스트 출력  
> -l : 상세 정보  
  -a : 숨김 파일  
  -S : 파일 크기순 정렬  

mkdir : 디렉토리 생성 
> -p : 부모 디렉토리 함께 생성  

touch : 빈 파일 생성  

cd : 디렉토리 이동  
> ../  : 상위 디렉토리 (/ 생략 가능)  
  ~ : 홈 디렉토리  
  / : 루트 디렉토리  

rm : 파일 삭제
> -r : 재귀적 삭제 (디렉토리 삭제)  
  -i : interactive
